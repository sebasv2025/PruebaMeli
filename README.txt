Proyecto: pruebaMeli

Reto técnico Mercado Libre donde se podrá identificar a traves de un vector dado si un humano es o no mutante con un API REST mediante un HTTP POST.

El stack tecnológico utilizado para el desarrollo de la aplicación fue:

Java - Spring, Maven, JPA - Hibernate, Postamn, Postgres.



Repositorio de la aplicacion en GitHub: https://github.com/sebasv2025/PruebaMeli.git


Instrucciones de uso aplicacion: pruebaMeli

1. En necesario realizar un fork del reporsitorio. El codigo se encuentra en el branch "Dev"
2. Debe crearse una base de datos Postgres donde se almacenarán los datos de los mutantes.
3. Puerto definido para 5432
4. La base de datos deberá llamarse "datos" 
5. Crear tabla "humano".
6. Generar petición POST con a través de: localhost:9091/mutant


Nota:El puerto de despliegue de la aplicación es el 9091.
